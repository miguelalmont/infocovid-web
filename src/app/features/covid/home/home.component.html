InfoCovid

Esta es una web informativa con datos actualizados diariamente.

Solamente utilizamos fuentes oficiales o confiables desde las que extraemos la información periódicamente de manera automática, minimizando la posibilidad de error.

Saber más

¿Qué es?

Nuestra web proporciona datos agregados de diferentes fuentes confiables para que puedas realizar el seguimiento del COVID-19 desde un único lugar.

¿Cada cuanto se actualiza?

Cada día obtenemos varios conjuntos de datos de diferentes instituciones, de esta manera, realizamos actualizaciones prácticamente cada hora con la nueva información.

¿De dónde sacamos los datos?

Todos los datos son oficiales a partir de los datos recopilados del Ministerio de Sanidad, del Dipartimento della Protezione Civile de Italia, Robert Koch Institute de Alemania, Santé publique France y de la Johns Hopkins University
